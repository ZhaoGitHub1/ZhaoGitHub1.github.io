IDEA快捷键总结

工具
  • 工具
  • 快捷键

IDEA快捷键总结：

参考文件：https://resources.jetbrains.com/storage/products/intellij-idea/docs/IntelliJIDEA_ReferenceCard.pdf

常用快捷键

描述 win快捷键 动图演示
全局搜索 Double Shift
快速修复 Alt + Enter
代码生成 Alt + Ins
最近打开文件 Ctrl + E
重命名 Shift + F6
全局搜索文本 Ctrl + Shift + F
按文件打开顺序返回/前进 Ctrl + Alt + Left/Right
提取代码块为方法 Ctrl + Alt + M

快捷键汇总

描述 win快捷键
打开/关闭对应的工具视图窗口 Alt + [0-9]
保存 Ctrl + S
同步 Ctrl + Alt + Y
最大化编辑器 Ctrl + Shift + F12
Inspect current file with current profile Alt + Shift + I
快速切换scheme Ctrl + BackQuote (`)
打开设置窗口 Ctrl + Alt + S
打开项目结构窗口 Ctrl + Alt + Shift + S
查找Action Ctrl + Shift + A

调试

描述 win快捷键
下一步，进入方法 F7
下一步，跳过方法体 F8
智能下一步，进入方法 Shift + F7
智能下一步，跳过方法 Shift + F8
Run to cursor Alt + F9
调出计算表达式窗口 Alt + F8
Resume program F9
新增/取消断点 Ctrl + F8
查看断点 Ctrl + Shift + F8

查询/替换

描述 win快捷键
全局搜索文件、命令等 Double Shift
搜索 Ctrl + F
搜索下一个/上一个 F3 / Shift + F3
替换 Ctrl + R
全局搜索文本 Ctrl + Shift + F
全局替换 Ctrl + Shift + R
选择下一个文本 Alt + J
选择所有匹配文本 Ctrl + Alt + Shift + J
取消选择文本 Alt + Shift + J

编辑

描述 win快捷键
Basic code completion Ctrl + Space
智能代码提示 Ctrl + Shift + Space
Complete statement Ctrl + Shift + Enter
Parameter info (within method call arguments) Ctrl + P
查看函数说明 Ctrl + Q
External Doc Shift + F1
Brief Info Ctrl + mouse
显示错误 Ctrl + F1
生成代码 Alt + Insert

重构

描述 win快捷键
复制文件 F5
移动文件 F6
安全删除 Alt + Delete
重命名 Shift + F6
呼出Refactor this弹窗 Ctrl + Alt + Shift + T
Change Signature Ctrl + F6
Inline Ctrl + Alt + N
提取代码块为方法 Ctrl + Alt + M
提取返回值为变量 Ctrl + Alt + V

导航

描述 win快捷键
搜索class文件 Ctrl + N
搜索文件 Ctrl + Shift + N
Go to symbol Ctrl + Alt + Shift + N
切换下一个/上一个tab Alt + Right/Left
激活前一个工具窗口 F12
返回编辑区 Esc
Hide active or last active window Shift + Esc
按行列坐标号跳转 Ctrl + G
最近打开文件 Ctrl + E

编译和运行

描述 win快捷键
编译项目 Ctrl + F9
Compile selected file, package or module Ctrl + Shift + F9
选择配置并运行/调试 Alt + Shift + F10/F9
运行/调试 Shift + F10 / F9
Run context configuration from editor Ctrl + Shift + F10

搜索

描述 win快捷键
项目和依赖库内查看调用处 Alt + F7
文件内查看调用处 Ctrl + F7
高亮本文调用处 Ctrl + Shift + F7
显示调用处 Ctrl + Alt + F7

版本控制

描述 win快捷键
本地提交 Ctrl + K
从远程仓库更新 Ctrl + T
推送到远程仓库 Ctrl + Shift + K
‘VCS’ quick popup Alt + BackQuote (`)

模板

描述 win快捷键
用模板环绕 Ctrl + Alt + J
插入模板 Ctrl + J

翻译插件Translation

描述 win快捷键
翻译选中单词/语句，浮窗显示 Ctrl + Shift + Y
弹窗打开翻译窗口 Ctrl + Shift + O
为所选单词提供替换建议 Ctrl + Shift + X

其他

描述 win快捷键
关闭IDE Alt + F4

快捷键和鼠标搭配

描述 win快捷键
跳转到方法的实现 Ctrl + Alt + 鼠标左键
跳转到方法的调用处 Ctrl + 鼠标左键
选择多行代码块 长按Alt +鼠标选择

快捷键提醒软件

Mac系统下推荐一款快捷键提示软件CheatSheet，长按Command(⌘)则会提示当前软件的快捷键，如下图为IDEA中快捷键提示
